# 递归算法
## 前言
    递归算法是一种编程技巧, 手段, 区别于分治, 动规, 贪心, 回溯, 这些是编程思想.
## 间接递归
    函数 f() 还可以通过一系列其他调用来间接地调用自身. 例如, 函数 f() 调用 g(), 函数 g()调用函数 f().
    eg: 
    f() -> g1() -> g2() -> g3() -> ... -> gm() -> f()
## 嵌套递归
```scala
// 样例
def h(n) = {
  n match {
    case 0 => 
    case x if x > 4 => n
    case _ => h(2 + h(2n))
  }
}
```
## 尾递归
    如果一个递归函数的递归调用语句是递归函数的最后一句可执行语句, 则称这样的递归为尾递归.
