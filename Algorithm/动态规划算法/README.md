#动态规划
## 一个模型,三个特征
    什么样的问题适合用动态规划来解决呢? 简单来说就是一个模型,三个特征.
### 一个模型: 多阶段决策最优解模型
    一个问题如果适合用动态规划解, 解决的过程中, 需要经历多个决策阶段,每个决策阶段都对应着一组状态. 然后我们寻找一组决策序列, 经过这组决策序列, 能够产生最终期望求解的最优值.  
### 三个特征
#### 1. 最优子结构(优化子结构)
    原问题的最优解一定包含子问题的最优解.我们可以通过子问题的最优解, 推导出问题的最优解.或者说后面阶段的状态可以通过前面阶段的送信
