2. 配合以下操作:
$ ssh-keygen -t rsa -P '' -f ~/.ssh/id_rsa
$ cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys // cat id_rsa.pub.29 >> authorized_keys
$ chmod 0600 ~/.ssh/authorized_keys // chmod 600 authorized_keys  //本地免密钥登录
