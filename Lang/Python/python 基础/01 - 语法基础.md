
## 语法基础
### 常用关键字
#### 常量
    True, False, None (相当于 java null)
#### 容器
    class, import, from, del
#### 判断
    if, elif, else, is, in, assert (单元测试用)
#### 循环
    for, while 
    continue：跳出当前循环
    break
    pass (什么都不做，语法有歧义的地方，用来作占位符用)
#### 异常
    raise, try, except, finally, as (自动关闭对象，比如文件句柄)
#### 运算符
    +, -, *, /, **, //,  %, >, >=, <, <=, ==, !=
#### 逻辑运算
    and, or, not
#### 位运算
    python 中没有异或
    >>, <<, &, |, ^ 
    二进制数: 0b1101
    print(0b1101 & 0b1011) # 0b1001 = 9
    
```python
# 求 n 是不是 2 的几次方
def is_pow2(n):
     return (n & (n - 1)) == 0
```
#### 其它
    缩进，注释
    多行代码：
```python
# et
str = 'abcd' \
      'efgh'
```
