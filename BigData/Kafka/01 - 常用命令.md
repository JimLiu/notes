* 创建 topic:
```BASH
${KAFKA_HOME}/bin/kafka-topics.sh \
--zookeeper x.x.x.x:2181 \
--create \
--topic $TOPIC_NAME \
--replication-factor 2 \
--partitions 3
```
* 消费数据:
```bash
${KAFKA_HOME}/bin/kafka-console-consumer.sh \
--new-consumer \
--bootstrap-server ip:9092 \
--topic $topic_name \
--max-messages 5  \
--from-beginning
```
* 删除 topic
```bash
${KAFKA_HOME}/bin/kafka-topics.sh \
--zookeeper zkip:2181 \
--delete \
--topic $topic_name
```
